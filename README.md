# E2_PFSD_TradEval

A Machine -learning integrated educational project for evaluating traditional trading strategies and performance.

## Overview

**E2_PFSD_TradEval** is a comprehensive platform designed to evaluate and analyze traditional trading strategies. This project demonstrates proficiency in full-stack development in python along with machine learning and is deployed and accessible at [https://e2-pfsd-trad-eval.vercel.app](https://e2-pfsd-trad-eval.vercel.app).
